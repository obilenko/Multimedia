Activities 5.3 - Multimedia

  1. Embed an audio file in an HTML page with the following requirements:
    - Player has controls to play, pause, seek, etc
    - Audio must automatically start playing when page loads
    - Audio must be muted when playback begins

  2. Create a HTML audio player that plays an audio file of your choice in Google Chrome, Internet Explorer and Mozilla Firefox. Use multiple sources if you have to.

  3. Create a HTML video player that plays a video file of your choice in Google Chrome, Internet Explorer and Mozilla Firefox. Use multiple sources if you have to.

  4. Create a WebVTT file for this video. Then embed this video in an HTML page with the captions you created.
